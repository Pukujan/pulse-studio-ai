  ## Running the code

  Run `npm i` to install the dependencies.

  **Add your API keys** to `.env` file (see [API_SETUP_GUIDE.md](./API_SETUP_GUIDE.md))

  Run `npm run dev` to start the development server.

  **Note**: The app works without API keys using fallback systems!
